# Brief_Triv-IA_Pursuit

## Instalation :

#### Create the virtual environment

- `cd /path/to/BRIEF_TRIV-IA_PURSUIT`
- `python -m venv .venv`
- Activate environment `source venv/bin/activate`

#### Init data base :

- `python database_tester.py`

#### Run sreamlit :

- `streamlit run main.py`

#### Play !
